Regras de Estados:
Os estados devem ser escritos de acordo com o padrão:
q"Estado Atual","Elemento Lido" -> q"NovoEstado","Elemento a Escrever","Movimento"
Exemplo: q1,1 -> q2,0,>

O estado inicial é obrigatório
O elemento inicial é obrigatório
Pelo menos um estado final deve ser informado
"q_" - Representa o estado inicial
"_" - Representa o elemento inicial
"$" - Representa um movimento da fita que é final
">" - Representa um movimento para direita
"<" - Representa um movimento para esquerda
"0" - Representa um elemento branco
